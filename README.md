# wpf-pinger

WPF project for check a list of ip address.

![alt text](http://patovega.com/github/imagenes/wpf-pinger/pinger.jpg)

# About this project

* WPF Project using C#
* .Net 4.6

### Run wpf-pinger 
You must add the xml file in the executable folder and add names and ip address childs to XML.
